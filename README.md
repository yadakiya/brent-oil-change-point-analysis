# Brent Oil Price Change Point Analysis

## Project Overview

This project analyzes historical Brent crude oil prices to determine how major geopolitical events, OPEC decisions, wars, sanctions, and economic crises influence oil prices.

The project applies Bayesian Change Point Detection using PyMC to identify structural breaks in the Brent oil price time series.

---

## Objectives

- Explore Brent oil price history
- Study statistical properties of the time series
- Detect structural breaks using Bayesian methods
- Associate detected change points with real-world events
- Build an interactive dashboard using Flask and React
